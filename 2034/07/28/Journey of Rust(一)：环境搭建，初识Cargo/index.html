环境安装

Rust的环境下载命令超级简单。在Terminal输入：

$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh

这个命令将下载一个脚本并按时安装rustup工具，这个工具将帮你安装rust的最新的稳定版本。可能会提示你输入管理员密码。

如果安装成功，将出现下面这个提示：

Rust is installed now. Great!

恭喜你，你已经完成Rust的环境安装啦。
